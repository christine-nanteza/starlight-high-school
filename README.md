# Starlight High School Management System

A PHP-based school management website with admin dashboard and MySQL database.

## Features
- Student and teacher management
- Admin dashboard
- Admissions system
- Marks and report generation
- Fee management
- Login and authentication system

## Built With
- PHP
- MySQL / phpMyAdmin
- HTML & CSS
- JavaScript
